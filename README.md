# Robotics Lab Screen

A fullscreen dashboard for the robotics lab TV.

## Live site

[Open the live display](https://YOUR-USERNAME.github.io/robotics-lab-screen/)

## What this is

This project is a static GitHub Pages site designed for a 16:9 4K display in the lab.  
It shows notices, events, and time/date information on a large always-on screen.

## Files

- `index.html` — page structure and content
- `style.css` — layout, colors, and TV-friendly sizing
- `app.js` — live clock and page behavior

## How to edit

1. Open the project files on your laptop.
2. Edit the content in `index.html`.
3. Edit styling in `style.css`.
4. Save your changes.
5. Commit and push to GitHub.

Example:

```bash
git add .
git commit -m "Update notices"
git push
```

## Deployment

This site is deployed with GitHub Pages.  
Changes pushed to the main branch are published to the live site URL.

## Display target

- Sony FW-85XD8501
- 3840×2160 resolution
- 16:9 aspect ratio
- Fullscreen browser display

## Notes

- Keep text large and readable from across the room.
- Avoid long scrolling pages.
- Prefer simple sections like notices, events, and QR links.
